## 👨‍💻 About

Personal portfolio website for **Vinayak Ichake** — Software Development Engineer at CAMS, M.Tech (CSE) from IIT Guwahati. Built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies, just clean frontend code.

---

## ✨ Features

- **Custom animated cursor** with lime-to-amber color shift on hover
- **CSS-only tech animations** — Matrix terminal, blockchain chain, neural pulse rings, HMM state diagrams, circuit board glow, audio wave bars, and network node graphs — all without external GIFs
- **Auto-scrolling GIF strip** that pauses on hover and reverses direction
- **Animated blob background** with morphing teal/lime/amber gradients
- **Scroll-triggered reveal animations** for every section
- **Photo placeholder** — click to preview your photo directly in the browser, or swap `src=` in HTML for permanent use
- **Active nav highlighting** on scroll
- **CRT scanline texture overlay** for that extra techie feel
- Fully **responsive** — works on mobile and desktop

## 🚀 Getting Started

### Run locally

```bash
# Clone the repo
git clone https://github.com/vinyl5200/PortFolio.git

# Open in browser — no build step needed
cd PortFolio
open index.html   # macOS
start index.html  # Windows
```

### Deploy to GitHub Pages

1. Push to the `main` branch
2. Go to repo **Settings → Pages**
3. Set source: `main` branch, `/ (root)` folder
4. Save — live at `https://vinyl5200.github.io/PortFolio/` in ~2 minutes

## 📸 Adding Your Photo

**Option 1 — Quick preview (browser only):**
Open `index.html` in a browser and click the photo placeholder box.

**Option 2 — Permanent (recommended):**
In `index.html`, find the `.photo-placeholder` div and replace it with:
```html
<img src="your-photo.jpg" alt="Vinayak Ichake" />
```
Then place `your-photo.jpg` in the same folder as `index.html`.
